# restaurant-menu-inheritance-java
## 📌 Project Description
This project demonstrates **Inheritance in Java** using a simple **Food Delivery Restaurant system**.

A base class **Restaurant** contains the restaurant name and a method to display the beverage menu.
Two subclasses **VegRestaurant** and **NonVegRestaurant** extend the Restaurant class and display their own special menus.

This program shows how **inheritance allows code reuse and specialization** in Object-Oriented Programming.

## 🏗️ Class Structure
Restaurant (Base Class)
│
├── VegRestaurant (Subclass)
│
└── NonVegRestaurant (Subclass)

## ⚙️ Features
* Display restaurant name
* Show beverage menu
* Veg restaurant special menu
* Non-veg restaurant special menu
* Demonstrates **Java Inheritance**
